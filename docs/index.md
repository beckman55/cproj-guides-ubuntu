# cproj-guides-ubuntu Documentation

Welcome to the documentation for **cproj-guides-ubuntu**, a clean, universal automation toolkit designed to make project creation, synchronization, and validation effortless on Ubuntu systems.

This repository provides:

- Universal project automation scripts (`cproj`, `newproj`)
- A full repository validator to enforce portability and cleanliness
- Installation and setup instructions
- GitHub Actions workflows for linting and validation
- A structured documentation system for future users
